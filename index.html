Get Started

Choose your path. The portable AppImage bundles everything — no dependencies needed. Or run from source with your own Python.

📦

Portable AppImage

Fully self-contained — bundles Python 3.11, PyQt6, and all dependencies. Download, make executable, run.

chmod +x GymBuddy-*.AppImage
./GymBuddy-*.AppImage ↓ Download Latest
What's inside
  • Python 3.11 + PyQt6 bundled
  • Zero system dependencies
  • Portable — runs from USB
  • Automatic updates via appstream
💻

From Source

Clone the repo, run the setup script, activate the virtual environment, and launch.

git clone https://github.com/connerjc234/gymbuddy.git
./setup.sh
source .venv/bin/activate
python -m workout_tracker.main View on GitHub
Requirements
  • Python 3.11+
  • PyQt6
  • Obsidian vault (optional)
  • Linux / macOS / WSL
